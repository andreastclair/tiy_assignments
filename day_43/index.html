Goodreads is a social site for readers and authors to find and share books that they love. With a Goodreads account, a user can build personal libraries with multiple shelves, track their reading progress, and find and recommend other titles to and from their friends.

Challenges

The current Goodreads site is too cluttered. There is a lack of overall hierarchy which makes navigation confusing. With all of the site's features displayed, it's difficult for the user to tell what's the most important and where their attention should go first. There is no clear difference in type treatment for links.

Goals

Define hierarchy on all pages. Make the site easier to navigate. Reduce clutter.

Solution

Use color and font-sizes to define hierarchy. Use the Gestalt principles to group like elements together, making the site easier to navigate. Combine elements into dropdown menus.

Grid

This project calls for a simple grid of one to two sections for easy navigation. In the small device grid, the single column layout allows for easy readability of dense content. In the medium and large devices, the layout shifts so that navigation options are in the 5 columns in the left section, with the majority of the content taking up the remaining seven columns on the right. All layouts are accompanied by an icon set in the footer for small devices and in near the header for the medium and large devices. All grids are designed with media queries for responsiveness.

Colors

My plan for the redesign is to focus on user achievements. Users will have the opportunity to gain badges when they complete books, sign up for new challenges, create new shelves, or participate in a reading poll. This new focus requires a set of colors that reflect positivity, encouragement, and winning. I've chosen bright colors with high contrast to create an inviting and exciting site that will engage the users and result in continuous use, motivating them to update their reading status and move on to the next book.

Here is the icon set for screen sizes larger than mobile, or, 700px. This set also shows how notifications will display.

The icons move from the foot of the pages and appear in the upper right corner as shown below.

The arrow icon is used for the "back to top" option on the shelves page so that users can jump back to the top of their shelf without having to scroll. The pencil icon is so that users can quickly and easily edit their shelves.

The Achievements icon will show on a users' profile page inside of the Achievements tab. This icon represents personal successes such as completing a book, reaching a milestone in their reading challenge, adding a shelf, participating in a poll, or joining Goodreads. This icon will have a 10px margin surrounding it on all sides and will be 33px / 57px.
